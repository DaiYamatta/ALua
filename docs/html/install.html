Instalação

Antes de iniciar a instalação certifique-se que os seguintes pré-requisitos estão instalados:

  • ptreads
  • OSSP uuid
  • Threading Building Blocks

A seguir serão indicados os passos para fazer a instalação do alua.

  1. Faça o download do ALua
  2. Extraia o arquivo com o comando:
    3. tar -xzvf alua.tar.gz
  3. Configure no arquivo configure as seguintes variáveis:
    4. # Include path
IPATH=-I. -I/usr/include -I/usr/local/include -I/opt/local/include
# Library path
LPATH=-L/usr/lib -L/usr/local/lib -L/opt/local/lib
# Lua library install path
LUA_INSTALL_LIB_PATH=/usr/local/lib/lua/5.1
# Lua install path
LUA_INSTALL_PATH=/usr/local/share/lua/5.1
  4. Compile e instale a biblioteca luasocket:
    5. $ cd luasocket
$ make
$ make install
  5. Execute o comando a seguir para carregar a biblioteca e verificar a instalação (não devem ser exibidas mensagens de erro):
    6. $ lua -l socket
  6. Compile e instale a biblioteca uuid:
    7. $ cd uuid
$ make
$ make install
  7. Execute o comando a seguir para carregar a biblioteca e verificar a instalação (não devem ser exibidas mensagens de erro):
    8. $ lua -l uuid
  8. Compile e instale a biblioteca rawsend:
    9. $ cd rawsend
$ make
$ make install
  9. Execute o comando a seguir para carregar a biblioteca e verificar a instalação (não devem ser exibidas mensagens de erro):
    10. $ lua -l rawsend
  10. Compile e instale a biblioteca ccr:
    11. $ cd ccr
$ make
$ make install
  11. Execute o comando a seguir para carregar a biblioteca e verificar a instalação (não devem ser exibidas mensagens de erro):
    12. $ lua -l ccr
  12. Compile e instale o alua:
    13. $ cd alua
$ make
$ make install
  13. Execute o comando a seguir para carregar a biblioteca e verificar a instalação (não devem ser exibidas mensagens de erro):
    14. $ lua -l alua
